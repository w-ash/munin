# Tier Pass

Working space for source-walking + tier assignment. **Deleted before the trip** with `obsidian delete` (sends to trash, recoverable). Deliberate exception to the vault-wide move-don't-delete rule: Pass files are pure scratch, so no creation date is lost.

- Write `tier`, `recommended_by`, and citations back to per-entry files (`<Category>/entries/`) — those are the durable record.
- Pass files hold the cross-tab + verdict during the walk; nothing else.
- Methodology meant to outlive trip cleanup → `Travel/<Trip>/References/Tier Process.md`, not the Pass README.

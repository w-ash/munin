# Ideas Conventions

## Required frontmatter

created: YYYY-MM-DD
tags:
  - idea
status: active | parked | done

## Structure

Keep Ideas docs concise:
- **Problem** — what's the gap or need
- **What we know** — research, options, findings so far
- **Open questions** — what's unresolved
- Link related Ideas to each other with wikilinks
